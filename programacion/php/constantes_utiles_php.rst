#####################
Constantes Utiles PHP
#####################

PATH_SEPARATOR
    En windows es ``;`` y en unix ``:``
    Útil por ejemplo para un ``set_include_path``

DIRECTORY_SEPARATOR
    En windows es ``\`` y en unix ``/``
    Útil para rutas

PHP_EOL
    End of line, lo mismo que ``"\n"``

PHP_OS
    Sistema donde se encuentra ejecutando php

PHP_VERSION
    Versión de PHP
