##################
Instalacion NodeJS
##################

Instalacion
***********

Fedora
======

.. code-block:: bash

    yum install nodejs

Ubuntu
======

.. code-block:: bash

    sudo apt-get install nodejs

Algunos programas
*****************

less
====

.. code-block:: bash

    npm install less -g

para usar lessc archivo.less > archivo.css

typescript
==========

.. code-block:: bash

    npm install typescript -g

para trnaformar tsc helloworld.ts
