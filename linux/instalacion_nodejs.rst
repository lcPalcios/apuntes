##################
Instalación NodeJS
##################

Instalación
***********

Fedora
======

.. code-block:: bash

    yum install nodejs

Ubuntu
======

.. code-block:: bash

    sudo apt-get install nodejs npm nodejs-legacy

Algunos programas
*****************

less
====

.. code-block:: bash

    npm install less -g

para usar: lessc archivo.less > archivo.css

typescript
==========

.. code-block:: bash

    npm install typescript -g

para transformar: tsc helloworld.ts

yuicompressor
=============

.. code-block:: bash

    npm install yuicompressor -g
