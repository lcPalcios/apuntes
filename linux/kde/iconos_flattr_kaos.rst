############################
Iconos Flattr de Kaos en KDE
############################

.. code-block:: bash

    cd ~/.local/share/icons
    git clone https://github.com/KaOSx/flattr-icons-kde.git

Ir a ``System Settings -> Application Appearence -> Icons`` y
seleccionar ``Flattr``
